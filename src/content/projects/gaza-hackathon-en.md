**Hackathon with a Cause - Hack for Gaza**

Participated in Hack for Gaza hackathon, developing an innovative offline messaging application that enables communication in crisis situations without requiring internet, WiFi, or SIM cards.

## Project Overview

GazaChat is an offline messaging application designed to address critical communication challenges in crisis situations. The app enables users to communicate even when traditional communication infrastructure is unavailable.

## Key Features

- **Offline Communication**: Enables messaging without internet, WiFi, or SIM cards
- **WebRTC Technology**: Leveraged WebRTC for peer-to-peer communication
- **LAN & Mesh Networks**: Implemented local area network and mesh network capabilities
- **React Frontend**: Built user-friendly interface using React
- **Crisis-Ready Design**: Specifically designed for emergency and crisis situations

## Technologies Used

- **WebRTC**: Peer-to-peer communication protocol
- **React**: Frontend framework for user interface
- **LAN Networks**: Local area network connectivity
- **Mesh Networks**: Distributed network architecture for resilient communication

## Impact

This project demonstrates innovative problem-solving for humanitarian technology, addressing real-world communication challenges in crisis situations where traditional infrastructure may be unavailable.
