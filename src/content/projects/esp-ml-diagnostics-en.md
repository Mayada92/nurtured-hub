**IPTC 2024 Research Project**

Evaluation of machine learning techniques for diagnosing Electrical Submersible Pump (ESP) events using synthetic time series data. This research demonstrates the effectiveness of ML approaches in identifying and classifying ESP operational issues.

## Project Overview

This project focuses on developing and evaluating machine learning techniques for automated ESP fault detection and diagnosis. The system uses synthetic time series data to train models that can predict equipment failures and optimize maintenance schedules.

## Key Features

- **Synthetic Time Series Dataset**: Created and utilized synthetic data for model training
- **Multiple ML Techniques**: Evaluated various machine learning approaches for ESP diagnosis
- **Fault Classification**: Developed models to identify and classify different ESP operational issues
- **Predictive Maintenance**: Contributed to improved predictive maintenance strategies
- **Industry Application**: Directly applicable to oil and gas operations

## Technologies Used

- **Machine Learning**: Various ML algorithms for classification and prediction
- **Time Series Analysis**: Processing and analysis of temporal data
- **Python**: Data science and ML implementation
- **Synthetic Data Generation**: Creating realistic training datasets
